Usage examples.
In the {@link it.trento.comune.j4sign.examples.CLITest} class you find a complete demonstration of the Smart card signature - CMS encapsulation - CMS verification process in a command line application. This can be easily transformed in a Swing GUI standalone application.

{@link it.trento.comune.j4sign.examples.PKCS11SignApplet} is a signature applet to be used in coordination with the server-side component {@link it.trento.comune.j4sign.cms.utils.CMSBuilder} which takes care of CMS envelope build and verification, see firma-digitale web application for a working example.
